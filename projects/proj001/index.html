Flappy Eggman

Flappy Eggman is a game based on the popular Flappy Bird but with a thematic twist featuring Dr. Eggman from the Sonic the Hedgehog franchise. The game is developed using standard web technologies, including HTML, CSS, and JavaScript.

Technologies Used:

HTML5: The game's structure is defined using HTML5, leveraging the semantic and multimedia capabilities of this version of the markup language.

CSS3: It is used to style and format HTML elements, providing an attractive and consistent presentation. The retro design is inspired by the graphics of classic Sonic games.

JavaScript (ES6+): The game's logic is implemented using JavaScript, taking advantage of the modern features of the language. The interactivity of Dr. Eggman, obstacles, and scoring logic is managed through this programming language.
